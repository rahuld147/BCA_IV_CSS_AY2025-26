**24BCA570P: Client Side Scripting**

**Course Objectives**

-   To know about basic understanding of JavaScript's.
-   To learn about control flow structures in JavaScript.
-   To learn JavaScript functions, objects, and arrays to solve problems.
-   To know about higher-order functions in JavaScript to achieve code abstraction.
-   To learn the fundamentals of objects in JavaScript using encapsulation, prototypes, classes, and inheritance.

**Course Outcomes**

1.  Understand JavaScript's basic elements, including values, data types, and operators.
2.  Apply JavaScript's program structure and control flow concepts to write code effectively.
3.  Analyze and apply higher-order functions for data transformation and abstraction in JavaScript.
4.  Evaluate JavaScript functions, data structures, and recursion to solve practical programming problems.
5.  Create JavaScript programs and classes, implementing object-oriented principles, inheritance, and polymorphism.

### High-3 Medium-2 Low-1

### Unit 1 12 Hours

**Basic Javascript Operations:** Introduction to JavaScript syntax, Performing arithmetic operations, Using built-in Math functions, variable manipulation.

### Unit 2 12 Hours

**Conditional Statements and Loops:** Using Math.random(),if-else statements to check conditions, modulus operator.

**Unit 3 12 Hours**

**String Manipulation and Functions:** Focuses on string manipulation and functions in JavaScript.

**Unit 4 12 Hours**

**Array and Date Operations:** Focuses on working with arrays and date operations in JavaScript.

**Unit 5 12 Hours**

**Advanced JavaScript Concepts:** Covers advanced JavaScript concepts and operations.
