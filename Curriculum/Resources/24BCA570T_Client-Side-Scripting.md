**24BCA570T: Client Side Scripting**

**Course Objectives**

-   To know about basic understanding of JavaScript's.
-   To learn about control flow structures in JavaScript.
-   To learn JavaScript functions, objects, and arrays to solve problems.
-   To know about higher-order functions in JavaScript to achieve code abstraction.
-   To learn the fundamentals of objects in JavaScript using encapsulation, prototypes, classes, and inheritance.

**Course Outcomes**

1.  Understand JavaScript's basic elements, including values, data types, and operators.
2.  Apply JavaScript's program structure and control flow concepts to write code effectively.
3.  Analyze and apply higher-order functions for data transformation and abstraction in JavaScript.
4.  Evaluate JavaScript functions, data structures, and recursion to solve practical programming problems.
5.  Create JavaScript programs and classes, implementing object-oriented principles, inheritance, and polymorphism.

### High-3 Medium-2 Low-1

**Unit-I: Introduction 5 Hours**

Java Script: Introduction, Values, Types, and Operators, Numbers, Strings, Boolean values, Unary operators, Empty values & Automatic type conversion.

**Unit-II: Program Structure 5 Hours**

Expressions and statements , Bindings, Binding names, the environment, Functions, the console.log function, return values, Control flow, Conditional execution , while and do loops, Indenting Code, for loops, Breaking Out of a Loop, Updating bindings succinctly.

**Unit-III: Functions & Data Structures 5 Hours**

Functions Defining a function, Bindings and scopes, Functions as values, Declaration notation, Arrow functions, the call stack, Optional Arguments, Closure, Recursion, Growing functions , Data Structures: Objects and Arrays, The were squirrel, Data sets, Properties, Methods, Objects, Mutability, Computing correlation, Array loops, The final analysis, Further arrayology, Strings and their properties, Rest parameters.

**Unit-IV: Higher-Order Functions 5 Hours**

Abstraction, Abstracting repetition, Higher-order functions, Script data set, filtering arrays, transforming with map, summarizing with reduce, Composability, Strings and character codes, recognizing text.

**Unit-V: Objects 5 Hours**

Encapsulation, Methods, Prototypes, Classes, Class notation, overriding derived properties, Maps, Polymorphism, Symbols, The iterator interface, Getters, setters, and statics, Inheritance, The instanceof operator.

**Self-Study Topics: 5 Hours**

Basic Computer Fundamentals, Client-Server Architecture, Understanding Web Browsers, HTML and CSS Basics, Document Object Model (DOM), Capitalization, Comments, Variables and Data Types, The Math Object, JSON, Conditional Statements, Switch Statements, Functions, Destructuring, Events and Event Handling, and JavaScript Applications.

**Total Hours: 30**
